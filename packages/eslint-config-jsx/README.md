Eslint config for jsx.

## Install

* npm

  ```bash
  npm install --save-dev @guanghechen/eslint-config-jsx
  ```

* yarn

  ```bash
  yarn add --dev @guanghechen/eslint-config-jsx
  ```

## Usage

* Use in .eslintrc.js

  ```javascript
  {
    extends: ['@guanghechen/jsx'],
  }
  ```


## Example

See [.eslintrc](https://github.com/guanghechen/node-scaffolds/tree/release-6.x.x/.eslintrc)

```json {5,29}
{
  "root": true,
  "extends": [
    "@guanghechen",
    "@guanghechen/jsx",
    "plugin:jest/recommended",
    "prettier"
  ],
  "plugins": ["import", "jest", "prettier"],
  "env": {
    "browser": true,
    "commonjs": true,
    "es6": true,
    "jest": true,
    "node": true
  },
  "rules": {
    "import/no-internal-modules": [
      2,
      {
        "allow": ["util"]
      }
    ]
  },
  "overrides": [
    {
      "files": ["**/*.ts", "**/*tsx"],
      "extends": [
        "@guanghechen/jsx",
        "@guanghechen/ts",
        "plugin:jest/recommended"
      ]
    }
  ]
}
```
